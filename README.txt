Project Setup
Clone the Repository
To get started with the project, follow these steps:

Opened terminal.
Ran the following command to clone the repository:
git clone https://github.com/Bismark-darfour/11286009_DCIT205

Installed Dependencies
Navigated to the project directory and install the necessary dependencies:
cd 11286009_DCIT205
npm install

Run the Application
Once the dependencies are installed, start the development server:
npm start

Visited http://localhost:3000 in web browser to access the application.
